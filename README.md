# Student Manager (Java + MySQL)

## 🚀 Features
- Add Student (Database)
- View Students
- Update Student
- Delete Student

## 🛠 Tech Stack
- Java
- JDBC
- MySQL

## 📌 Description
This is a menu-driven CLI application built using Java and MySQL.
It performs full CRUD operations by connecting Java with a database using JDBC.

## 📈 Learning Journey
- Started with basic Java classes
- Added user input using Scanner
- Implemented ArrayList-based storage
- Built menu-driven system
- Integrated MySQL using JDBC
- Converted to full database-driven application

## 💡 Key Concepts
- OOP (Classes, Objects)
- JDBC
- PreparedStatement
- SQL (CRUD operations)
